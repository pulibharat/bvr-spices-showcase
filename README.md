# BVR Spices Showcase

This is the official showcase application for BVR Spices, featuring our premium collection of spices.

## Project info

This project is built with:
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

To run this project locally:

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Deployment

Build the application for production:
```sh
npm run build
```
